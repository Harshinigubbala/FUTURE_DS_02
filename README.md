# Customer Retention & Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn patterns, retention trends, and customer behavior using Power BI. The dashboard helps identify factors influencing customer churn and provides insights to improve customer retention.

## Tools Used
- Power BI
- DAX
- Data Visualization

## Key Metrics
- Customer Count: 7K
- Churn Rate: 26.54%
- Average Monthly Charges: 64.76
- Average Total Charges: 2.28K
- Average Tenure: 32.37 Months

## Dashboard Visualizations
1. Customer Count by Churn
2. Contract Type vs Churn
3. Internet Service vs Churn
4. Churn Rate KPI
5. Monthly Charges KPI
6. Total Charges KPI
7. Average Tenure KPI

## Key Insights
- Month-to-month customers have the highest churn rate.
- Customers with one-year and two-year contracts are more likely to stay.
- Fiber optic customers show higher churn compared to other internet service types.
- Long-tenure customers are less likely to churn.

## Recommendations
- Encourage customers to switch from month-to-month plans to longer-term contracts.
- Introduce loyalty rewards for long-term customers.
- Investigate reasons behind higher churn among fiber optic users.
- Develop targeted retention campaigns for high-risk customers.
